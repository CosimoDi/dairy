# 个人交往红线

什么样的人可能会让我感到厌烦（讨厌这个人）：

1. 过于强调泊来的知识和体验，炫耀其通过短期或长期努力达成的成就。
   注：自己在生活、实践、阅读等过程中，感受到的体验是很好的。经过自己的经验，和某些理论有共鸣是好的。自己想做的事，做成了和我分享喜悦是好的。
   以炫耀为目的，在交流过程中“自己的感受“和“对方的感受“都缺位，是非常不好的。比喻一下就是，举着一个破木头说这个是圣物，我是神谕者，你们快来崇拜我·····
2. 对大部分事情认为“理所当然“，既没有敬畏态度，也没有深度求索的好奇心。
3. “社会达尔文主义“支持者，疑似部分人属于边缘型人格天生不具备共情能力（这部分人可以理解），但大部分社达只是因为生活苦恼而把怨恨外包，极度以自我为中心。
4. 攀比苦难，觉得这个世界所有人经历的创伤都比不上他经历的。

什么样的事件及反应会让我回避和这个人的关系（无关是否讨厌他）：

背景：我的“自我“目前是一块一块的，状态之间差异非常大，过去和现在可能差异也非常大。

1. 对方不理解，也不接纳这种自我状态，认为这种情况就是“想的太多“或者“吃饱了没事干“。
2. 可能是某一部分自我对这个人会有冒犯，如果这次冲突无法解决，那我会远离这个人。
3. 对方是我不知道怎么面对的人，或者是他们的朋友之类，聊天过程中会经常触碰到我很混乱的记忆。
4. 如果我们原本关系尚可，但对方觉得我很烦，且不与我沟通，我会快速抽离。
