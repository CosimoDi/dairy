# 柯3.23

怎么一下子就是 23 号了，因为周五在做 gateway， 周六和周日转瞬即逝（周六很晚才开始，周日很早就结束）

产品工作的好处是，其实越投入越能触及到一些深入的部分。

最近在逛 x，觉得这的确是一个非常好的社区，比什么其他国内的知识类社区或者兴趣类社区都好，同时满足了我很多欲望？而且里面的人类疑似是比较前卫的，非常好观察，这样我无聊的时候可以选择刷 x，非常充实啊，因为能看到一个人几个月的脉络。

做了两版方案和一个规划，其实觉得 gateway 没什么必要提，主要和毛老师提 v2 和规划复盘就好了。我的交付频率的话，大概是一周迭代一次 demo？可以暂时部署在我的服务器上给别人测试。

比较重要的思考就是商业化的部分，这部分难且有价值····公司的意义所在吧。其实觉得大部分的公司都有让社会变好的愿景在，不如说没有这方向愿景（起码明面上）的公司很多被淘汰了吧。更好的满足人的欲望某种程度上也算是让社会变好。

其实赡养一部分人类完全可行吧，生产力大解放的时候，少部分人创造价值、享受更稀缺的体验，大部分人只是去体验别人的作品。其实这样对双方都好，目前很多人也是处于大部分这个阶段吧，不过还是希望自己成为少数人。

之前的体验是，感受能力是区分“同类“的标志，但后面这种想法又一次被否定了。因为差异化的人但有救赎的体验会非常好？更多的长出自己的东西吧，用那种肆意的体验吃掉那些好的作品。

3:50醒来的，觉得凌晨除了没有吃的东西+不能用电之外，完全是很好的时间。以后独居能不能过上晚上八点睡，凌晨四点醒来的生活呢？其实觉得天慢慢亮比天慢慢黑更浪漫，或者更舒适吧，苏醒的感觉是不错的。

大概七点钟的时候，和信表达了一些关于平等友谊的事情，会希望她有自我的空间之类的。嗯，其实是觉得如果她也是先有一个自我的话，那就是很好的，我们的关系更不容易越界吧。
