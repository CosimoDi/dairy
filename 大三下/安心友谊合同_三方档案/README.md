# 安心友谊合同三方档案

这份档案按飞书《安心友谊合同》的组件骨架整理，本地只额外保留一层“运行与修订”支持层。

## 顶层结构

- 00_合同正文：当前合同正文和 2.20 草案入口。
- 01_规划同步：按月整理柯、桃两条线的推进记录，并收纳规划输入与背景材料。
- 02_双方红线：边界与回避条件。
- 03_经验分享：双方教学与主题分享。
- 04_聊天记录：信件、访谈、转写和会议入口。
- 05_运行与修订：三方同步、修订记录、复盘输入等不直接属于飞书原始组件的本地支持材料。

## 当前整理原则

- 顶层以飞书组件为准，不再并列拆出“日志”“个人规划”“复盘”三类目录。
- 个人规划输入下沉到 01_规划同步/90_规划输入。
- 背景性日志和阶段上下文下沉到 01_规划同步/99_背景材料。
- 复盘、同步和结构修订下沉到 05_运行与修订。
- 聊天记录继续放在 04_聊天记录，并补进实际镜像文件。
- 密码、恢复码和其他凭据不纳入这套档案。
- GitHub 仓库默认只推这套档案，不推整个 Diary 工作区。
