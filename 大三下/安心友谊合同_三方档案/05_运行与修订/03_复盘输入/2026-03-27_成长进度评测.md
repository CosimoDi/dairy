# 背景：
    3.24号和毛老师聊复盘规划的时候，发现自己对自己的成长定位不够清晰，于是补充学习了《俞军产品经理方法论》（注：该作品发布于 2022 年）关于产品经理潜力划分、成长曲线、能力水平、跳槽成本等知识，给自己更明确的定位。

    1. 高阶产品经理的衡量标准和他所做的产品高度相关
    2. 合适的产品实践机会永远是最稀缺的。
    3. 标准 sp 产品的进阶路线：先积累 3 年以上都基本功，再到以下六个方面做取舍和锻炼：从 0 到 1，做深做透，做宽做杂，大用户量，商业闭环，团队建设
    4. 深度思考能力+共情能力 是 A 类产品的标准
    5. 方向的跃迁难度取决于产品的智慧水平，一个年薪 200 万的产品脱离平台可能只能有 50 万年薪的产出。但系统思考能力强的产品（标准是半年内快速学习业务知识）是可以跃迁的，只不过这部分人非常少、且换方向通常 ROI 不高（解释了之前问毛老师的职业变迁，毛老师主要在聊决策的现象）。

# 自评：
    1. 目前我的长板在于深度思考能力、一定的批判性思维和一定的技术理解、共情部分有所缺失。具体体现在，毛老师给一个理想回答案例后，我能根据此拆出全面的回答逻辑并跑出较理想的 mvp。但我写不出那个理想案例，主要还是对我们的目标用户理解不够充分。
    2. 当前的实践模式，本质是我和毛老师有一定磨合后，基本功训练与产品深度实践并行，能力锻炼属于：从 0 到 1，商业闭环两个部分，正式上线、乃至用户量扩大到一定水平之后、有机会锻炼其他几个方面能力。
    3. 由于时代区别，目前一定的技术理解重要性被放大，学习技术变成 ROI 很高的事情，故需要同步推进。

# 结论
我理解的产品经理能力稀缺性及价值排序：

产品定义与产品决策 > 项目方案设计与项目管理 > 具体的功能设计、产品设计、需求开发

大厂路线进阶缓慢的原因：资源过饱和。
产品定义、产品决策、方案设计，本质上都只需要少数人参与其中，且这部分人才的价值是最大的；校招时，只需要多花几万块、筛出一批 sp，让他们做执行、同时给一定的培养，然后再从 sp 里筛出可能有参与到前置步骤中的，就非常的赚了。没有这种人才也没关系，因为用户量大招一些做精细化的设计和运营的人也是不错的。

为什么我能参与到前两个阶段中：

公司层面：
1. 问问大象公司处于起步阶段，产品定义到方案设计的绝大部分是毛老师做，有一定人才缺口。

3. 毛老师的用人偏好：因为毛老师的职业发展属于兴趣、价值驱动，对同样有这类特质的下属会更关注，也更善于发挥这类人的潜力。

个人层面：
1. 我的能力培养总体上依照问问大象的培养体系，加上之前跟毛老师共事过，初期阶段很快适应了。

3. 过去对于创业领域的探索、过去对通识知识的积累，让我能很快理解公司处于什么阶段。
