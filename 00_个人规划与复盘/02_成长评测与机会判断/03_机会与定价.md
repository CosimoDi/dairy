## 毛老师信息客观性

**结论**：大概率真诚，但真诚与公司利益高度一致。

- **不是纯白嫖**：高质量反馈 + 框架输入 + 角色授权均真实存在
- **不是纯培养**：你的产出正服务于公司最重要的产品探索

- **最大风险**：创始人单点判断 = 整个制度。长期只接受内部评价会高估外部可兑现程度

**毛老师背景**：百度 → 去哪儿 → 内涵段子 → 洋葱学院 → 百词斩 → 问问大象。覆盖大厂 + 独角兽 + 字节系 + 教育深耕，方法论经过多场景实战。这增强了"他给的训练内容有质量"的可信度。

## 市场定价模型

你是 27 届 211 AI 策略产品方向。定价是五层叠加：

| 层 | 内容 | 你的情况 |
|----|------|----------|
| 校招批发价 | 学历 + 城市 + 预算 | 211 本科，不是短板 |
| 岗位稀缺溢价 | AI 产品 vs 普通产品 | 方向明确，懂 Prompt/工作流/评测/Agent |
| 高潜溢价 | 值不值得更高档招入 | 做过超纲内容，有空间 |
| 风险折扣 | 无大厂/无量化结果/无锚点 | 三项均缺，会被折 |
| 角色错配补偿 | 已做超纲的事 | 取决于对方是否信经历可迁移 |

**你更像"AI 产品高潜校招"，不是"成熟社招产品"。** 不要按"像几年社招"定价，要按"校招基准上浮几档"想。

## 问问大象定价机制

比标准校招多一层，因为创始人强决策：

1. **市场下限**：同类 AI 产品校招能拿多少
2. **创始人判断**：毛老师觉得你值不值得押
3. **替代成本**：你走了多久能找到替代
4. **路线绑定**：你是不是早期必须保留的人

前两项定区间，后两项定杠杆。

## 转正报价红线

以下 = 偏低：

1. 按"普通实习转正"给，不体现 AI 产品方向
2. 不体现你参与价值验证和商业模式构建
3. 只口头承诺，无明确 title / 薪资结构 / 复盘节点
4. 理由围绕"你还是学生"而非替代成本

合理报价至少包含：

- **岗位 title**：AI 产品经理 / AI 策略产品（不是模糊实习延长）
- **固定薪资**：按校招正式员工谈
- **复盘节点**：3/6 个月调薪重评机制
- **角色边界**：是否继续参与产品定义、评测、商业模式
- **股权/期权**：如果现金有限，股票期权是一种价值匹配方式
- **实质放权**：产品 Owner 的事实授权，不只是口头信任

## 跳槽时展示的核心能力

### 已验证（有内部证据）

1. **AI 对话产品工作流设计** — 多 agent→单场景收敛、状态机 + Prompt + 检索 + reasoning
2. **AI 产品评测体系** — 测试集、分类、回归链路
3. **价值验证与产品定义** — 参与"值不值得立项"判断
4. **角色自觉** — 业务目标→策略→工作流→评测连成完整岗位

### 对外表达原则

- 不说"我在创业公司做了很厉害的项目" → 说"我负责把咨询逻辑转成 AI 产品工作流，并建立评测与回归体系"
- 不主打"毛老师说我是 SP" → 主打具体做了什么、遇到什么难点、怎么拆解和验证

### 需要准备

1. **脱敏 portfolio**：背景→目标→状态机→评测→失败样本→优化
2. **面试话术**：为什么这样拆工作流、怎样定义成功标准、模型 vs Prompt vs 检索的责任划分
3. **角色证明清单**：参与了什么决策、定义了什么标准、推动了什么变化
