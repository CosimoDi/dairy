## 2. 替代成本

| 维度 | 事实 |
|------|------|
| 你走后谁接 | **没人**。方向上只有你一个产品 |
| 外部招人替代周期 | 至少 6 个月+（需同时补业务 + 技术） |
| 你自己上手花了多久 | 1 个月，但当时定位是泛挖掘，现在已是战略项目 |
| 公司招过同类角色吗 | 未招过 |
| 同类人选多吗 | 毛老师没直接说，但事实上很少 |
| 文档 vs 默契 | 默契 > 文档。文档可写，但读文档到会做事要很久 |

**替代成本结论**：短期无法替代，中期（6 个月+）才可能培养出替代者。这是你最硬的谈判杠杆之一。

## 5. 创始人视角变量

这些是毛老师作为创始人定价时，他自己大概率会权衡的：

| 变量 | 对他的含义 |
|------|-----------|
| 现金流 | 小公司现金流能承受多少固定人力成本 |
| 你的不可替代窗口 | 现在最依赖你的阶段是什么时候（立项期？产品化期？上线期？） |
| 他对市场价的认知 | 他觉得 AI 产品校招值多少，可能和你的认知有差距 |
| 绑定意愿 | 他是想长期留你，还是只需要你做完当前阶段 |
| "第一批人给高出发点"的兑现 | 他自己说过这话，你到时候可以拿来对齐 |

## 谈判策略含义

你的立场是“想留，但必须价值匹配”。这意味着：

1. **不是“不给钱就走”**——你愿意接受多元化补偿（放权、期权、title）
2. **也不是“无论如何都留”**——报酬与价值不匹配是硬红线
3. **核心谈判姿态**：我想和你一起把这个产品做起来，但我需要看到你对我价值的认可体现在结构里
4. **可接受的补偿组合**：固定薪资 + 期权/股票 + 明确放权（事实上的产品 Owner title）+ 定期评估节点
