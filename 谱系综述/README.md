## 写作约定

- 依据 DSM-5-TR 与 ICD-11 主流分类，同时列出两者差异。
- 证据等级以现行国际循证指南（NICE、APA、CANMAT/ISBD、WFSBP、RANZCP 等）为优先引用源。
- 临床数据给出粗略患病率、性别比、起病年龄、共病图谱与预后指标。
- 不替代临床诊疗，所有用药与处置需在合格专科医生指导下进行。
